Ollama setup guide
1

Install Ollama

ollama.com/download →
2

Enable Chrome extension access Required

Add to your shell config (one-time setup):

echo 'export OLLAMA_ORIGINS="chrome-extension://*"' >> ~/.zshrc && source ~/.zshrc

Then restart Ollama to apply:

pkill ollama && ollama serve

Without this, Ollama will reject requests from the extension (403).

3

Pull a model

ollama pull llama3.2
